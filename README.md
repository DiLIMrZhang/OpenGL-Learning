## Learning OpenGL on Xcode

### 安装glfw
1、在终端中运行命令`brew install glfw3`

glfw将会被安装在`/usr/local/Cellar/glfw`

2、配置路径

在`/usr/local/Cellar/glfw`路径下找到`header`和`lib`对应路径，在`Xcode->Preferences->Localtion->Custom Paths`设置路径。
